## ✨ 이번 디벨롭으로 들어간 핵심 메커니즘

**선택지 시스템 (4지선다 재설계)**
- 모든 시나리오가 기존 상/중/하 3개 → **균형형(8점) 1개 추가로 4지선다**.
- **확률형 보기(도박)**: 일부 「즉효」 보기는 성공/실패가 갈리는 도박. (예: 지하수 무제한 양수, 남획, 동토 시추, 지구공학 강행)
- **지연 청구서**: 값싼 선택이 지금은 멀쩡해도 N턴 뒤 청구서로 폭발. (석탄 재가동·매립지 확장·화학비료 등)
- **티핑포인트 결합**: 생태계가 낮을수록 기온 피해가 비선형으로 증폭.

**내러티브**
- 시나리오마다 **피해자의 목소리(화자)** 한 줄.
- **상태 인지형 콜백**: 과거의 선택(석탄·외면·무모한 도박)이 이후 결과 화면에 회상으로 되돌아옴.

**UX 연출**
- 본문 **타이핑 효과** → **선택지 순차 등장(읽기 강제)** → **커밋 애니메이션**.
- 기온이 오를수록 **화면 전체가 붉게 달아오르는 색온도 드리프트**.
- **하드코어 모드**: 선택 **시간압박 링**(시간 초과 시 최악의 선택 강제 집행).

**라이브옵스 (반복 플레이)**
- **엔딩 도감**(12종 수집), **숨겨진 「대전환」 엔딩**(완벽한 플레이에서만 등장).
- **시너지 업적**: 완벽주의 통제관(전 단계 최선), 운명의 도박사(도박 3회 성공).
- **🗓️ 데일리 챌린지**: 날짜 시드로 전 세계가 같은 판 — 친구와 점수 경쟁.

**난이도 4종**: 🔥도전 · ☠️하드코어 · 💰예산 · 🗳️정치
- **예산/정치 모드**의 전략적 타협: 환경을 포기하는 하/중 선택지는 오히려 **예산·정치자본을 회복**시켜 줍니다. 예산이 바닥나면 "이번 턴은 눈 딱 감고 자금 확보" 같은 선택이 가능. (예산 모드 난이도도 완화)

## 🚀 GitHub Pages 배포 (3단계)

> 이 폴더는 push만 하면 자동 배포되도록 GitHub Actions 워크플로우(`.github/workflows/deploy.yml`)가 들어 있습니다.

### 1) GitHub에 새 저장소를 만들고 코드를 올립니다
이 폴더에서 터미널을 열고:

```bash
git init
git add .
git commit -m "feat: 선택지 시스템 디벨롭(4지선다·도박·지연·도감·데일리)"
git branch -M main
git remote add origin https://github.com/<your-username>/<repo-name>.git
git push -u origin main
```

### 2) Pages 소스를 GitHub Actions로 지정
저장소 → **Settings → Pages → Build and deployment → Source** 를 **GitHub Actions** 로 선택.

### 3) 끝
push가 끝나면 **Actions** 탭에서 배포가 돌고, 1~2분 뒤
`https://<your-username>.github.io/<repo-name>/` 에서 게임이 열립니다.
이후엔 코드를 고쳐 `git push` 할 때마다 자동 재배포됩니다.

### (선택) GitHub CLI가 있다면 한 번에
```bash
gh repo create <repo-name> --public --source=. --remote=origin --push
```
그 다음 Settings → Pages → Source를 **GitHub Actions** 로 바꿔주세요.

## 📁 구조
```
index.html   # 마크업 + 스타일(인트로/시나리오/엔딩 카드)
data.js      # 순수 데이터 — 시나리오·선택지·엔딩 규칙·내러티브 콜백 (로직 없음)
main.js      # 게임 로직 — 선택 파이프라인·사운드·돌발변수·업적·저장·공유 카드
.github/workflows/deploy.yml  # Pages 자동 배포
```

> 설계 원칙: **콘텐츠는 `data.js`에 선언적으로, 로직은 `main.js`에 제네릭하게.**
> 새 시나리오·선택지·엔딩은 `data.js`에 한 줄씩 추가만 하면 `main.js`는 그대로 동작합니다.
